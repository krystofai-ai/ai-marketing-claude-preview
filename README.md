# AI Marketing Toolkit pro Claude Code

Řekl jsem Claudovi co chci — a vznikl celý marketingový tým v terminálu.

Jedno velení. 13 skillů. 5 specializovaných agentů pracujících paralelně.

`/market audit` → audit webu, SEO, CRO, brand, konkurence, strategie — vše najednou.

---

## Co to dělá

```
/market audit https://tvůj-web.cz
```

→ 5 agentů analyzuje tvůj web současně  
→ Každý skóruje svoji dimenzi (obsah, konverze, SEO, konkurence, brand)  
→ Výsledek: kompletní `MARKETING-AUDIT.md` s prioritizovanými opravami

---

## 13 příkazů, jeden toolkit

| Příkaz | Co dostaneš |
|---|---|
| `/market audit` | Celý marketing audit — 5 agentů paralelně |
| `/market seo` | On-page + technické SEO + připravenost na AI vyhledávání |
| `/market landing` | Audit konverzního poměru landing page |
| `/market brand` | Brand hlas, archetyp, tónová matice |
| `/market copy` | Přepis textů v brand hlase |
| `/market emails` | E-mailové sekvence (welcome, nurture, launch…) |
| `/market social` | Balíček příspěvků pro více platforem |
| `/market ads` | Reklamní texty pro Meta, Google, LinkedIn, TikTok |
| `/market competitors` | Konkurenční inteligence + poziční mezery |
| `/market funnel` | Mapa funnelu + specifikace GA4 eventů |
| `/market launch` | 4týdenní launch plán |
| `/market proposal` | Klientská nabídka s kalkulačkou cen |
| `/market report` | Klientský report z auditů |

## Jak to vzniklo

Tohle není výsledek měsíců programování.

Je to výsledek toho, že víš **jak správně komunikovat s AI** — jak strukturovat skilly, jak spouštět agenty paralelně, jak dostat přesně to co chceš.

Kdybych ti dal zdrojový kód bez kontextu, budeš stuck na prvním problému.
